# dotfiles
dotfiles

## Outside configuration
1. Install a nerd font to support symbols. For mac I installed [Ubuntu Sans Mono Nerd Font Mono](nerdfonts.com)
2. Install aerospace window manager [AeroSpace Install](https://nikitabobko.github.io/AeroSpace/guide#installation)
3. Install jq to dynamically discover bracket items in sketchybar (and for like a 100 other things) `brew install jq`
